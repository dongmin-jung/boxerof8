[id="deleting-a-project-using-the-web-console_{context}"]
= 웹 콘솔을 통해 프로젝트 삭제하기

{product-title} web console을 통해 프로젝트를 삭제할 수 있다.

[NOTE]
====
권한 없으면 *Delete Project* 옵션을 사용 못한다.
====

.과정

. *Home* -> *Projects* 으로 가라.

. Options menu {kebab}에서 *Delete Project* 를 선택해라.
